# docs/scaffold/

This project was seeded from the [docks](https://github.com/DocksDocks/docks) cross-tool plugin scaffold — its structure (context-tree `AGENTS.md`/`CLAUDE.md` node pairs, dual plugin manifests, bundled plan-lifecycle skills, author-side validators) mirrors that baseline.

`docs/scaffold/` is reserved for capturing *this* repo's own shape into a reusable `spec.yaml` + `templates/` (via the `scaffold` skill's `setup` mode), so new projects can be seeded from `effect-kit` in turn. It holds only this node pair until that capture runs.
